# EduFun Educational Website

## Project Overview

EduFun is a multi-page educational website designed to provide information about different learning areas such as AI & ML, Data Science, Web Development, and Graphic Design. The main aim of this project was to create a simple and user-friendly online learning platform using HTML and CSS.

This project was developed as part of my web development coursework. Through this project, I practiced webpage structure, navigation, layout design, styling, responsive design, and basic user interface presentation.

---

## Website Pages

The website includes multiple pages such as:

- Home Page
- Courses Page
- AI & ML Page
- Data Science Page
- Web Development Page
- Graphic Design Page
- Instructors Page
- Testimonials Page
- FAQ Page
- Contact Page

---

## Features

The main features of the website are:

- Multi-page website structure
- Navigation bar for moving between pages
- Course category pages
- Instructor section
- Testimonials section
- FAQ section
- Contact form layout
- Card-based course design
- Images and visual sections
- Hover effects
- Basic responsive layout
- Clean and simple user interface

---

## Technologies Used

- HTML
- CSS
- Visual Studio Code

---

## Purpose of the Project

The purpose of this project was to understand how a complete website is planned and developed. Instead of creating only one webpage, I tried to build a small educational platform with different pages and sections.

This project helped me understand:

- How to structure HTML pages
- How to use CSS for styling
- How to create navigation between pages
- How to design course cards and content sections
- How to maintain a consistent layout across multiple pages
- How to organize website files and images
